# A simple ToDoList

## 📋 Description

A simple ToDoList with React.

- Create(Add new task)
- Read(Task view)
- Update(Edit task)
- Delete(Delete task)

## 🛠️ Technologies & Requirements

- Node.js (20.6)
- React (18)

## 🚀 Getting Started

1. Install the packages

```
$ npm install
```

2. Run the server

```
$ npm run dev
```

3. Open http://localhost:3000 to view it in your browser.
